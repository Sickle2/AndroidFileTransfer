Android 文件传输

Mac 与 Android 手机之间最简单的文件传输方案。无需安装手机 App，WiFi 扫码直传。

⬇️ 下载 macOS 版 查看源码

macOS 12+ · Apple Silicon / Intel Universal · 免费开源

核心功能

📶

WiFi 无线传输

手机扫描二维码，在手机浏览器中直接浏览、下载 Mac 上的共享文件，或把手机文件上传到 Mac。无需安装任何 App。

🔒

可控共享范围

默认只共享你手动添加的文件和文件夹，隐藏文件自动过滤，手机端看不到真实的 Mac 路径。上传统一进入接收目录。

🔌

ADB USB 传输

连上 USB 数据线，即可在应用中浏览手机的 /sdcard，双向推送和拉取文件，适合需要稳定高速传输的场景。

🖥️

原生 macOS 体验

基于 Wails 构建，Go 后端 + React 前端，体积小、启动快，原生窗口交互，支持 Apple Silicon 和 Intel Mac。

🗂️

文件夹内浏览

手机端支持多级目录导航，有面包屑和「← 上级」按钮，操作像本地文件管理器一样直观。

⚙️

配置持久化

共享文件列表、接收目录等配置自动保存，重启应用后无需重新设置。配置文件存放在 Application Support 目录。

下载安装

  1. 前往 Releases 页面，下载 AndroidFileTransfer-x.x.x.dmg。

  2. 打开下载的 DMG 文件，把 AndroidFileTransfer.app 拖入右侧的 Applications 文件夹。

  3. 在 Launchpad 或 Applications 文件夹中打开应用。

  4. 确保手机和 Mac 在同一个 WiFi 网络下，在设备列表中选中 WiFi 设备，扫描二维码即可开始传输。

⚠️ 提示："无法验证开发者" / "Apple 无法检查是否包含恶意软件"

本应用未经 Apple 付费公证，macOS Gatekeeper 会在首次启动时拦截。这是正常现象，请按以下任一方法放行：

方法一（推荐）

在 Finder 找到 AndroidFileTransfer.app，右键单击 → 打开，弹窗中点击"打开"。之后双击正常启动。

方法二

系统设置 → 隐私与安全性，在底部"已阻止使用 AndroidFileTransfer"旁点击"仍要打开"，输入密码确认。
